## About

A Homebrew repo to install Waktu Solat CLI

## Install

```sh
brew untap waktusolatmy/tools &>/dev/null || true
brew tap waktusolatmy/tools

brew install waktusolat
waktusolat version
```

## Uninstall

```sh
brew uninstall waktusolat
brew untap waktusolatmy/tools
```
